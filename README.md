# Express MongoDB API

## Instalación
* Modificar el archivo .env y añadir tus credenciales

```bash
npm install
```

## Ejecutar tests

`npm run test` o `npm run test:watch`

## Linter

He utilizado el linter [standardjs](https://standardjs.com/)

* `Seed` Cargar datos en la Base de datos
  ```sh
  npm run seed:news
  ```
